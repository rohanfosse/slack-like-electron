Feature "Cahier" : editeur WYSIWYG collaboratif temps reel pour groupes projet CESI (3-5 etudiants).

**Why:** Les etudiants quittent Cursus pour Google Docs des qu'ils doivent rediger ensemble. Le Cahier les garde dans l'app.

**How to apply:**
- Stack: TipTap (ProseMirror) + Yjs + y-prosemirror + Socket.IO existant + SQLite
- Integre au canal projet, a cote des documents (pas de module separe)
- MVP: editeur WYSIWYG + curseurs colores + sauvegarde auto. Pas d'historique, pas d'export.
- Spec complete: `.claude/specs/deep-interview-cahier-collaboratif-2026-04.md`
- Decide le 13 avril 2026 via deep interview (13 rounds, 87% clarte)
